Usage

Optional parameters are surrounded by [brackets]

Multiple choices for an input are separated by |

The following commands will work with Jarvis present in the channel:

.HelenBot

  • Usage: .HelenBot
  • Displays the current version of the bot.

.hugHelen | .helenHug

  • Usage: .hugHelen
  • Hugs the bot!
  • Warning: she may respond with a snarky comment.

.hugme

  • Usage: .hugme <response for .hugHelen>
    • .hugme Hello, friend.
  • Sets the message you'd like Helen to say when you hug her.
  • Warning: only specific users are allowed to do this.

.pronouns | .pronoun

  • Usage: .pronouns <username>
    • .pronouns Secretary_Helen
  • Displays the pronouns of a user.

.myPronouns

  • Usage: .myPronouns
  • Displays the pronouns of the person who uses the command.

.clearPronouns

  • Usage: .clearPronouns
  • Clears the pronouns of the person who uses the command.

.setPronouns

  • Usage: .setPronouns [accepted] <list of pronouns>
    • .setPronouns she her hers
    • .setPronouns accepted they them their
  • Saves your preferences for regular or accepted pronouns.

The following commands only work if Jarvis is not present.

.au

  • Usage: .au [<username>]
    • .au
    • .au HelenBot
  • Displays stats about an author on the Foundation wiki.
  • If no username is specified, displays stats about the person who uses the command.

.ch | .choose

  • Usage: .ch <comma-separated list of choices>
    • .ch Be a bot, Be a human
  • Picks one of several choices.

.def | .definition

  • Usage: .def <query>
    • .def revolver
  • Searches the Merriam-Webster dictionary for a term.

.g | .google

  • Usage: .g <query>
    • .g How is HelenBot so awesome?
  • Searches Google for a term.

.gis

  • Usage: .gis <query>
    • .gis revolver with a smiley face
  • Searches Google Images for a term.

.lc | .lastcreated

  • Usage: .lc
  • Lists the three most recently created pages on the Foundation wiki.

.mode

  • Usage: .mode
  • Displays the currently active mode: Enabled for everyone, or Administrators only.

.msg

  • Usage: .msg <user> <message>
    • .msg Secretary_Helen You're doing a great job!
  • Sends a message to a user.

.roll

  • Usage: .roll <amount>d<sides> [<bonus>] [-e] [<message>]
    • .roll d20 + 4 - 2d6
    • .roll 3dF - 20df
    • .roll 1d20 +2 Attack the gazebo
    • .roll 1d20 + 2 -1dF -e Flee from the gazebo
  • Rolls one or more dice.
  • -e expands the result to show each result up to 20.
  • dF are fudge dice.

.myRolls

  • Usage: .myRolls
  • Displays the last five rolls made by the person who uses the command, expanded.

.avg | .average

  • Usage: .avg <dice sides>
    • .avg 20
  • Displays the average of your dice rolls with the specified number of sides.
  • Warning: if you roll a preposterously high number of dice at once, like 500d6, only the first 20 rolls will be saved to Helen's database for calculating your average. Maybe don't roll that many dice at once!

.seen

  • Usage: .seen [-f] <username>
    • .seen Secretary_Helen
    • .seen -f Secretary_Helen
  • Displays the most recent message seen from a user, along with the time it was sent.
  • -f displays the first message seen instead.

.s | .sea | .search

  • Usage: .s <query> [-a <author>] [-t <tag>] [-e <exclude>] [-u]
    • .s emulator
    • .s emulator -t electronic
    • .s -a HelenBot -t tale -t anderson -u
  • Searches for articles on the Foundation wiki.
  • -u displays a summary of all articles found instead of listing individual results.
  • -e and -t can be used multiple times.

.sm

  • Usage: .sm <number>
    • .sm 2
  • Selects one of the options given by Helen as part of a "Did you mean:".

.tell

  • Usage: .tell <user> <message>
    • .tell Secretary_Helen You're doing a great job!
  • Records a message to send to a user the next time Helen sees them.

.w | .wiki | .wikipedia

  • Usage: .w <query>
    • .w SCP Foundation
  • Searches Wikipedia for a term.

.y | .yt

  • Usage: .y <query>
    • .y Never gonna give you up
  • Searches YouTube for a term.

.user

  • Usage: .user <username>
    • .user HelenBot
  • Provides a link to an author's Wikidot profile.

SCP-XXXX

  • Usage: SCP-<article>
    • SCP-3209
    • SCP-Big-egg-J
  • Displays the SCP article with the corresponding designation.

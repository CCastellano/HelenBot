Usage

Optional parameters are surrounded by (parentheses)

Parameters with choices are shown with a | symbol i.e. either or

Parameters with [square brackets] are NOT optional

The following commands will work with Jarvis present in the channel:

.HelenBot

  • Arguments: None
  • Effect: Displays the current version of the bot.

.hugHelen|.helenHug

  • Arguments: None
  • Effect: Hugs the bot! Warning: she may respond with a snarky comment.

.hugme

  • Arguments: [response for .hugHelen]
  • Effect: Sets the message you'd like Helen to say when you hug her. Warning: only specific users are allowed to do this.
  • Example: .hugme Hello, friend.

.pronouns|.pronoun

  • Arguments: [username]
  • Effect: Displays the pronouns of a user.
  • Example: .pronouns Secretary_Helen

.myPronouns

  • Arguments: None
  • Effect: Displays the pronouns of the person who used the command.

.clearPronouns

  • Arguments: None
  • Effect: Clears the pronouns of the person who used the command.

.setPronouns

  • Arguments: (accepted) [space separated list of pronouns]
  • Effect: Saves your preferences for regular or accepted pronouns.
  • Example: .setPronouns he him his
  • Example: .setPronouns accepted they them their

The following commands only work if Jarvis is not present.

.au

  • Arguments: ([username])
  • Effect: Displays stats about an author on the Foundation wiki. If no username is specified, displays stats about the person who used the command.
  • Example: .au
  • Example: .au HelenBot

.ch|.choose

  • Arguments: [comma separated list of choices]
  • Effect: Picks one of several choices.
  • Example: .ch Be a bot, Be a human

.def|.definition

  • Arguments: [query]
  • Effect: Searches the Merriam-Webster dictionary for a term.
  • Example: .def revolver

.g|.google

  • Arguments: [query]
  • Effect: Searches Google for a term.
  • Example: .g How is HelenBot so awesome?

.gis

  • Arguments: [query]
  • Effect: Searches Google Images for a term.
  • Example: .gis revolver with a smiley face

.lc|.lastcreated

  • Arguments: None
  • Effect: Lists the three most recently created pages on the Foundation wiki

.mode

  • Arguments: None
  • Effect: Displays the current active mode (Enabled for everyone, or Administrators only)

.msg

  • Arguments: [user] [message]
  • Effect: Sends a message to a user.
  • Example: .msg Secretary_Helen You're doing a great job!

.roll

  • Arguments: [amount][d|f][sides] (bonus) (-e) (message)
  • Effect: Rolls one or more dice. -e expands the result to show each result up to 20. dF are fudge dice.
  • Example: .roll d20 + 4 - 2d6
  • Example: .roll 3dF - 20df
  • Example: .roll 1d20 +2 Attack the gazebo
  • Example: .roll 1d20 + 2 -e Flee from the gazebo

.myRolls

  • Arguments: None
  • Effect: Displays the last five rolls made by the person who used the command, expanded.

.average|.avg

  • Arguments: [dice sides]
  • Effect: Displays the average of your dice rolls with the specified number of sides. Warning: if you roll a preposterously large amount of dice at once, like 500d6, only the first 20 rolls will be saved to Helen's database for calculating your average. Maybe don't roll that many dice at once!
  • Example: .avg 20

.seen

  • Arguments: (-f) [username]
  • Effect: Displays the most recent message seen from a user, along with the time it was sent. -f displays the first message seen instead.
  • Example: .seen Secretary_Helen
  • Example: .seen -f Secretary_Helen

.s|.sea|.search

  • Arguments: [query] (-a [author]) (-t [tag]) (-t [another tag]) (-e [exclude]) (-e [another exclude]) (-u)
  • Effect: Searches for articles on the Foundation wiki. -u displays a summary of all articles found.
  • Example: .s emulator
  • Example: .s emulator -t electronic
  • Example: .s -a HelenBot -t tale -t anderson -u

.sm

  • Arguments: [number]
  • Effect: Selects one of the options given by Helen as part of a "Did you mean:".
  • Example: .sm 2

.tell

  • Arguments: [user] [message]
  • Effect: Records a message to send to a user the next time Helen sees them.
  • Example: .tell Secretary_Helen You're doing a great job!

.w|.wiki|.wikipedia

  • Arguments: [query]
  • Effect: Searches Wikipedia for a term.
  • Example: .w SCP Foundation

.yt|.y

  • Arguments: [query]
  • Effect: Searches YouTube for a term.
  • Example: .y Never gonna give you up

.user

  • Arguments: [username]
  • Effect: Provides a link to an author's Wikidot profile.
  • Example: .user HelenBot

SCP-XXXX

  • Arguments: None
  • Effect: Displays the SCP article with the corresponding designation.
  • Example: SCP-3209
  • Example: SCP-Big-egg-J
